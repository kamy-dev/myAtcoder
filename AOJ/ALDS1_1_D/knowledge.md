## 復習時に見直すメモ

[Link](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_1_D)

- Rj−Ri where j>i

## 減少するパターンも考える必要がある

```bash
5
5
4
3
2
1
```

この問題は、`j>i` という条件があるので、1回は取引をしなければならないので、最大値が-1 となる。

また、
```bash
4
3
1000
5
1
```

だと、997 が最大。

## ポイント

- maxv の値を、Rt の制約から、最も小さい値 ( 例)`- 10**18` ) にすること。2回目の操作から、前の時点と今の時点のRj-Riの単純差分が計算される。
- minv の値を、最も大きい値 ( 例)`10**18` ) にすることで、forループの度に最も小さい値が入る。なので、1, 3, 4 と最大利益のタイミングに間隔が空いても問題ない。
