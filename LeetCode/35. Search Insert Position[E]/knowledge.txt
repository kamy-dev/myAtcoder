・O(log n) の計算量とは
    -> 二分探索(binary search)

    具体的な説明は以下↓
    https://note.com/scg_tech/n/n19bdab62747f?utm_source=chatgpt.com

    二分探索が主な O(log n) のアルゴリズム。
    二分探索を使う場合の制約は
        ①対象がリストである
        ②リストがソート済み
    の２つ。

    具体例) 要素が64のリストで二分探索　-> 64個のデータを6回の探索でデータを探せる。
    1回目：リストの中央値と探す値を比較。残り32個
    2回目：32個の中央値と探す値を比較。残り16個
    3回目：16個の中央値と探す値を比較。残り8個
    4回目：8個の中央値と探す値を比較。残り4個
    5回目：4個の中央値と探す値を比較。残り2個
    6回目：2個の中央値と探す値を比較。1つに絞れる。

    式にすると、64 / 2 / 2 / 2 / 2 / 2 / 2 = 1
    この式を一般化すると、
    N / 2^n = 1
    N = 2^n
    となり、これが対数log_2 N = n となるため、二分探索はO(log n)であると言える。
